# 🍕 FoodIt – Responsive Food Ordering Website

FoodIt is a **responsive food ordering web application** built using **HTML, CSS, and JavaScript**, designed for a single restaurant/seller.
It provides a clean, modern UI and simulates a real-world food ordering experience with cart functionality and dynamic interactions.

---

## ✨ Features

### 🎨 UI/UX Design

* Modern, clean, and user-friendly interface
* Smooth animations and hover effects
* Card-based food menu layout
* Optimized for readability and usability

---

### 🛒 Core Functionality

* Browse food menu items
* Add/remove items to cart
* Update item quantity dynamically
* Real-time price calculation
* Simple checkout interface

---

### 📱 Responsive Design

* Fully responsive across:

  * Mobile devices 📱
  * Tablets 📱
  * Laptops & desktops 💻
* Built using flexible layouts (Flexbox & Grid)

---

## 🧩 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla JS)** – Interactivity & logic
